# OpenAI Frontier

Frontier team at OpenAI, the group working on the most advanced applications of AI agents. Ryan Lopopolo's organization driving the harness engineering and dark factory research.

## Key Projects

- **Extreme Harness Engineering**: Building autonomous coding agent pipelines that process 1B+ tokens/day
- **Dark Factory Software Pipeline**: 0 lines of human-written code, 10+ PRs/engineer/day
- **Symphony Orchestration**: Multi-agent workflow system (Elixir/BEAM backbone)
